# login-system
This is a simple registration and login system created using PHP and MySQL. It allows users to sign up for an account, login, and logout.

#Features
User registration: users can sign up for an account by providing their name, email, and password.
Password hashing: passwords are hashed using the password_hash() function to ensure security.
User login: registered users can login to their account using their email and password.
Session management: user sessions are managed using the $_SESSION superglobal variable to keep users logged in until they logout.
Logout: users can logout of their account at any time.

#Requirements
PHP 7.0 or higher
MySQL 5.5 or higher

#Installation
Clone the repository to your local machine.
Import the users.sql file into your MySQL database.
Configure the database connection in config.php.
Upload the files to your web server.

#Usage
To register a new account, navigate to the registration page and provide your name, email, and password.

To login to your account, navigate to the login page and provide your email and password.

To logout of your account, click the "Logout" button.
